## 🧬 Tentang Saya

```yaml
nama: LeoMods
fokus: WhatsApp Bot API • Telegram Bot Framework • Website Development
bahasa: Bahasa Indonesia 🇮🇩
status: Beginner developer, masih terus belajar
motto: "Kalau bisa diotomatisasi, kenapa dikerjain manual?"
```

Saya adalah developer pemula yang sedang menekuni dunia bot automation dan pengembangan web. Belum lama terjun ke sini, tapi belajarnya serius dan konsisten — ngoprek project kecil-kecilan setiap hari sampai paham betul cara kerjanya. Berikut area yang lagi saya dalami:

**🤖 WhatsApp Bot API**
Saya belajar membangun bot WhatsApp multi-device pakai library **Baileys** di atas Node.js. Mulai dari bikin command handler dasar, auto-reply, sampai fitur tambahan seperti sticker maker, downloader media, dan mini-game berbasis teks. Masih terus belajar soal session handling dan struktur command yang rapi, tapi bot yang sudah jadi cukup stabil dipakai harian.

**📨 Telegram Bot Framework**
Selain WhatsApp, saya juga eksplorasi bot Telegram pakai **Telegraf.js**. Fokus belajarnya di pesan interaktif — inline keyboard, rich message, sampai struktur project yang gampang dikembangin lagi ke depannya. Pelan-pelan mulai paham alur webhook dan polling.

**🌐 Website Simple**
Untuk sisi web, saya bikin website ringan — biasanya portofolio pribadi atau landing page kecil — dengan sentuhan animasi dan tema gelap yang aesthetic. Belum kompleks, tapi terus dipoles dari sisi tampilan maupun struktur kodenya.
